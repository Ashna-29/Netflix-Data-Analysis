# 🎬 Netflix Data Analysis

## 📖 Project Overview

This project explores the Netflix Movies and TV Shows dataset using Python to identify trends in content distribution, genres, ratings, countries, release years, and movie durations. The project demonstrates the complete exploratory data analysis (EDA) workflow, including data cleaning, analysis, visualization, and interpretation of results.

---

## 🎯 Objectives

The analysis aims to answer the following questions:

* How many Movies and TV Shows are available on Netflix?
* Which countries produce the most Netflix content?
* Which genres are the most common?
* How has Netflix's content changed over the years?
* What are the most common content ratings?
* What is the distribution of movie durations?

# 🔍 Key Findings

* Movies make up a larger portion of Netflix's catalog than TV Shows.
* The United States contributes the highest number of Netflix titles.
* Netflix's content library experienced substantial growth after 2015.
* Drama, International Movies, and Comedies are among the most common genres.
* TV-MA is one of the most frequent content ratings.
* Most Netflix movies have durations between approximately 90 and 120 minutes.
